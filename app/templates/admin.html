Admin: Archive Root Settings

Use this page to rotate archive Drive root folders before hitting the 500k-file limit. Updates apply without restarting the app.

In Shared Drive, create a folder and name SMARTSHEET_BACKUP_{Number e.g. 2}, then copy the ID in the URL after drive/folders/{"THIS ID"}. Paste that folder ID in the form below and click Save Runtime Settings.
