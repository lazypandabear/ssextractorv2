Service Account Setup

The app uses SmartSheetDataArchive.json from the project root. Make sure the Google Drive parent folder is shared with the service account below:

  1. Share the parent folder with smartsheet-data-archiving@jira-onboarding.iam.gserviceaccount.com.
  2. Confirm the service account has Editor access so it can create folders and upload files.
